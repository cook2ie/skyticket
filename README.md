# ✈️ SkyTicket Web Re-Design (Personal Project)

기존 항공권 예매 사이트를 바탕으로,  
정보 전달 중심의 구조는 유지하면서도  
보다 직관적이고 감각적인 분위기로 재해석한 리디자인 프로젝트입니다.

## 🧩 Overview

- 항공권 예매 사이트를 주제로 한 웹 리디자인 프로젝트
- 정보량이 많은 화면을 보다 직관적으로 정리한 UI 구성
- 예약/검색 흐름을 고려한 레이아웃 재구성
- 전체적으로 더 미니멀하고 깔끔한 비주얼 톤으로 재해석

---

## 🛠 Tech Stack

- HTML / CSS / JavaScript
- jQuery

---

## 👨‍💻 Key Points

- 기존 사이트의 정보 구조를 바탕으로 사용 흐름이 더 자연스럽게 이어지도록 화면 구성
- 검색, 예약, 프로모션 영역의 우선순위를 정리해 가독성 향상
- 복잡하게 느껴질 수 있는 여행/예매 UI를 보다 깔끔하고 직관적인 레이아웃으로 재해석
- 전체적인 비주얼 톤을 정리해 보다 세련되고 몰입감 있는 화면으로 구성

## 💡 What I Learned

- 정보량이 많은 예매 사이트일수록 시각적인 분위기보다 먼저 정보 구조와 사용자 흐름을 정리하는 것이 중요하다는 점을 배웠습니다.
- 리디자인 작업에서는 단순히 화면을 새롭게 보이게 만드는 것보다, 사용자가 더 쉽고 빠르게 정보를 이해할 수 있도록 구조를 개선하는 과정이 중요하다는 점을 경험했습니다.
- 검색과 예약처럼 목적이 분명한 UI는 감성적인 비주얼뿐 아니라, 기능 흐름이 자연스럽게 이어지도록 설계해야 완성도가 높아진다는 점을 느꼈습니다.
